# como baixar ISO Windows 10

- O que é ISO?<br>
Um arquivo que contem **uma copia exata** de um sistema de arquivos de um disco ópitico (CD/DVD).<br>
permite instalar um o SO (sistema operacional) em uma maquina física ou virtual.
